Course Overview

Systems Administration: This course teaches students how to configure, manage, and maintain server systems in various IT environments. It covers operating systems (Windows, Linux), user account management, networking, virtualization, and system security. Students learn to set up and troubleshoot servers, implement access controls, and automate administrative tasks using scripting. The course also introduces cloud-based solutions and best practices for maintaining system reliability and performance. By the end of the course, students will be prepared to manage IT infrastructures in business and enterprise settings.

Noteworthy Assignments/Projects

  • Linux Server Configuration: Set up a Linux-based server with proper security policies and user access management.
  • Cloud Deployment: Deploy a server application on AWS or Azure and optimize it for performance.
