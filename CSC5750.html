Course Overview

Principles of Web Technology: This course introduces students to web development tools, frameworks, and best practices for building modern websites and web applications. Topics include HTML, CSS, JavaScript, responsive design, and front-end frameworks like React or Angular. The course also covers back-end technologies, including server-side scripting, APIs, and databases. Students gain hands-on experience by developing dynamic, interactive websites while learning about web security, accessibility, and performance optimization. The goal is to equip students with the skills needed to create functional and user-friendly web applications

Noteworthy Assignments/Projects

  • Responsive Portfolio Website: Build a personal portfolio with HTML, CSS, and JavaScript.
  • Full-Stack Web Application: Develop a dynamic web app using a front-end framework and a back-end API.
